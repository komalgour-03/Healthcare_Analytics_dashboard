# 🏥 Healthcare Analytics Dashboard

## 📌 Project Overview
This project is a Healthcare Analytics Dashboard developed using **MySQL** and **Power BI**. It helps analyze hospital operations by tracking patient appointments, revenue, doctors, departments, and payment status through interactive dashboards.

---

## 🎯 Objectives
- Analyze patient and appointment data.
- Monitor hospital revenue.
- Track appointment status (Completed, Cancelled, No Show).
- Compare payment status (Paid vs Pending).
- Provide interactive filters for better analysis.

---

## 🛠️ Tools & Technologies
- MySQL
- Power BI
- SQL
- CSV Datasets

---
